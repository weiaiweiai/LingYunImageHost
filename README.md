# LingYunImageHost

这是一个图床程序，是使用C# 开发的

本人是做winform的不太会做blazor程序，还请大家见谅

如果遇到问题欢迎大家提交 issues

如果有新需求，大家也可以提交issues

## 部署需要什么？

1、安装.net（运行环境）

2、安装nginx（反代理）

## 已实现的功能？

1、管理员账号密码（默认账号：Admin 默认密码：admin123@）

2、批量上传图片

3、配置页

4、自定义图片存储路径

## 待实现的功能？

1、多数据库支持，MYSQL，SQL SERVER ，等

2、用户管理

3、持续优化界面
